This is an undergraduate major project on emotion recognition based voice chatbot using transformer model. 
The NLP project is based on transformer model that generates replies based on emotion and intent of the users.

 
Scripts/ contains all pretrained models loaded and used in the project. 
Web application is built using JS for frontend and Fast API for backend.

- run-- npm start
- run-- uvicorn API:app--reload

Refer https://blog.tensorflow.org/2019/05/transformer-chatbot-tutorial-with-tensorflow-2.html 
 for more details on chatbot with transformer model.
